Weekly Numbers
Weekly Momentum Snapshot

How to Use This Tool

This app turns your weekly numbers into the Weekly Momentum Check format used by the GSD skill. Fill it in once a week, generate, and paste into Claude.

When to run it

Run this at the start of your weekly GSD review. Use the prior week's totals: bids submitted, average response time, win rate, calls completed, LinkedIn posts, and profile comments.

Where the numbers come from

  • Bids Submitted / Win Rate / Calls — from App 03 Bid Pipeline Dashboard.
  • Avg Response — average days between bid submission and first reply, across the week's bids.
  • LinkedIn Posts / Profile Comments — your own count for the week.
  • Top Case Study / Strongest Platform — whichever produced the best result this week.
  • Recommendation — one sentence on what to adjust next week based on the numbers above.

Using the output

Click Generate Snapshot, then Copy to Clipboard. Paste the result into Claude Code to start your GSD review session.

Where the data lives

Your entries are saved in this browser so the form is not empty next time you open it. Nothing is sent anywhere else.
